Problem Statement

Core Problems

  • High fatalities due to delayed alerts
  • GPS inaccuracies in India (~100m errors in challenging terrain)
  • Dependency on cellular/subscriptions
  • False positives in detection
  • Alert flooding

Specific Gaps

  • Legacy vehicles lack integration
  • MQTT connectivity issues (Wi-Fi/firewall dependencies)
  • Limited indigenous tech use
  • No geo-decimation for redundancy

Objectives

Primary

Develop a NAVIC-based retrofit crash detection system for accurate, subscription-free alerts

Secondary

Implement publish-subscribe via MQTT for multi-responder delivery; introduce geo-decimation for alert optimization

Long-term

Enable classifier models for better detection; scale to advanced NAVIC chipsets for <2.5m accuracy

Link to real-world impact: reducing 180,000 fatalities annually

Scope of the Project

In-Scope

Out-of-Scope

Proposed Methodology & System Architecture

Methodology

  • Multi-sensor fusion (vibration + inertial) for detection
  • NAVIC for positioning
  • BLE to app
  • MQTT pub/sub for alerts
  • Geo-decimation (geohash clustering)

Architecture Steps

  • Threshold-based logic for crashes
  • JSON payloads with coordinates/medical info
  • Hardware stack → BLE → Android Publisher → MQTT Broker → Subscribers

Conclusions

Addresses Critical Gaps in Road Safety

This project confronts key challenges in vehicular emergency response, focusing on the limitations of conventional GPS, the high cost of commercial systems, and the lack of safety features in legacy vehicles.

Proposes an Innovative, Indigenous Solution

The core of this project is the design of a novel framework that integrates India's NAVIC satellite system for superior regional positioning. This is combined with a multi-sensor fusion approach for reliable crash detection and a modern publish-subscribe architecture for scalable alert delivery.

Establishes a Foundation for Significant Impact

The prototype development validates a cost-effective and retrofit-compatible approach that has the potential to make automated crash detection accessible to a much wider range of vehicles. This work lays the groundwork for a scalable solution that can enhance national road safety and promote technological self-reliance.
